# KochList

Eine Progressive Web App (PWA), die per Sprach- oder Texteingabe eine Einkaufsliste für ein Gericht erstellt.

## Funktionen

- Spracheingabe auf Deutsch
- Automatische Erkennung von Gericht und Personenanzahl
- Einkaufsliste mit Mengenangaben
- Basiszutaten ausblenden
- Weiterleitung zu REWE und EDEKA
- Offline-fähig (Service Worker)
- Installierbar als PWA

## Unterstützte Gerichte

- Pasta Carbonara
- Pasta Bolognese
- Kartoffelbrot
- Zwiebelkuchen
- Frikadellen mit Kartoffelsalat

## Lokale Entwicklung

Einfach `index.html` in einem lokalen Server öffnen (z. B. mit VS Code Live Server).

## Deployment

Das Projekt ist statisch und kann direkt auf Vercel deployt werden — kein Build-Schritt erforderlich.
